## Resumo

Este trabalho foi desenvolvido como parte de uma atividade proposta pelos orientadores do **Americas TechGuard**, tendo como base técnica o artigo **"U-RNN: High-Resolution Spatiotemporal Nowcasting of Urban Flooding"**, de **Cao et al. (2025)**, seu repositório oficial e o conjunto de dados **UrbanFlood24 Lite**. O objetivo foi estudar a arquitetura do modelo U-RNN, compreender seu funcionamento e reproduzir sua etapa de inferência para previsão de inundações urbanas em curto prazo (*nowcasting*).

Como parte da atividade, foram realizadas adaptações para possibilitar a execução do modelo em um computador Windows da biblioteca da instituição, utilizando exclusivamente CPU e pesos pré-treinados, devido à ausência de GPU dedicada. Ao final, foram analisados os resultados obtidos, as métricas de desempenho, as limitações da infraestrutura utilizada e o potencial de aplicação do U-RNN em soluções de monitoramento e alerta para inundações urbanas no contexto do Americas TechGuard.
