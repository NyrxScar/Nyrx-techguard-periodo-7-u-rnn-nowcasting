# U-RNN Tutorials · 教程

Step-by-step reproduction guides for **[U-RNN](../README.md)** — high-resolution spatiotemporal nowcasting of urban flooding.

The project [README](../README.md) gives the quick overview and a 2-minute browser demo. These tutorials are the full, hand-holding walkthrough — pick a language and follow the recommended order.

**Recommended order:** Setup (1 → 3) → Inference (4) → Training (5). No local GPU? See 6. Reference & troubleshooting in 7–8.
